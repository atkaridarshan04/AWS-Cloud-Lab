Understanding S3 & CloudFront

Amazon S3 (Simple Storage Service) is an object storage service that allows you to store static files (HTML, CSS, JavaScript, images, etc.). When you enable static website hosting on S3, it serves your files via an HTTP endpoint.

Amazon CloudFront is a Content Delivery Network (CDN) that distributes your website across multiple AWS edge locations, improving speed and performance by serving content from the closest location to the user.
